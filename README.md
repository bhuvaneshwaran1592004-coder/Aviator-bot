# 🎮 Aviator Game Telegram Bot

Live Aviator round production bot with real-time multiplier tracking and betting system!

✅ **Live Round Updates** - Current round information  
✅ **History Tracking** - Last 10 rounds view pannu

## 🚀 Setup Instructions

### Step 1: Install Python
- Download Python 3.8+ from [python.org](https://www.python.org/downloads/)

### Step 2: Clone Repository
```bash
git clone https://github.com/bhuvaneshwaran1592004-coder/aviator-bot.git
cd aviator-bot
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Get Telegram Bot Token

1. Open Telegram app
2. Search **@BotFather**
3. Send `/newbot`
4. Follow instructions
5. Copy bot token

### Step 5: Configure Bot Token

Open `config.py`:
```python
TELEGRAM_TOKEN = "8690680893:AAFlpBS8WmoNA8_gRnT_eqdg-uOcIuWiafk"  # Replace with your token
```

### Step 6: Run Bot
```bash
python bot.py
```
